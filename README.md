# MediScholar AI

## AI-Powered Multi-Agent Medical Research Platform

MediScholar AI is a secure, full-stack, multi-agent AI platform that assists researchers, clinicians, and students in conducting evidence-based medical research. It leverages Retrieval-Augmented Generation (RAG), autonomous AI agents, and medical literature analysis to generate structured, citation-backed research outputs.

## Tech Stack

- React
- Tailwind CSS
- FastAPI
- LangGraph
- LangChain
- FAISS
- Supabase
- OpenAI
